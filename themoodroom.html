The Mood Room

A JavaScript driven web app that lets users express different moods through interactive text animations and sound effects.

Project Overview

The Mood Room is an interactive web application I conceptualized to explore emotions through playful digital interactions. Users can select a mood and type their thoughts, which then animate on the screen with corresponding sound effects. I designed all “pages” within a single index.html for simplicity and seamless interaction. The project has potential to evolve into a therapy or mental health focused tool for self expression and mood tracking. This was my first project focusing heavily on JavaScript, giving me hands on experience with DOM manipulation, event handling, audio integration, and creating a dynamic, user focused interface.
